Class: Animate

Animate

new Animate ()

each DOM element has its own Animate object returned by add() or accessed through HTMLElement.__domEase
Fires:
  • Animate#event:each-*
  • Animate#event:complete-*
  • Animate#event:loop-* - called when animation repeats or reverses

Events

complete-*

fires when there are no more animations where name is the name of the element being animated (e.g., complete-left fires when left finishes animating)
Type:
  • Animate

each-*

fires on each loop where there are animations where name is the name of the element being animated (e.g., complete-left fires when left finishes animating)
Type:
  • Animate

loop-*

fires when an animation repeats or reverses where name is the name of the element being animated (e.g., complete-left fires when left finishes animating)
Type:
  • Animate
