Class: DomEaseElement

DomEaseElement

new DomEaseElement ()

each DOM element has its own DomEaseElement object returned by add() or accessed through HTMLElement.__domEase
Fires:
  • DomEaseElement#event:each-*
  • DomEaseElement#event:complete-*
  • DomEaseElement#event:loop-* - called when animation repeats or reverses

Members

element HTMLElement

element being animated

Events

complete-*

fires when there are no more animations where name is the name of the element being DomEaseElementd (e.g., complete-left fires when left finishes animating)
Type:
  • DomEaseElement

each-*

fires on each loop where there are animations where name is the name of the element being DomEaseElementd (e.g., complete-left fires when left finishes animating)
Type:
  • DomEaseElement

loop-*

fires when an animation repeats or reverses where name is the name of the element being DomEaseElementd (e.g., complete-left fires when left finishes animating)
Type:
  • DomEaseElement
